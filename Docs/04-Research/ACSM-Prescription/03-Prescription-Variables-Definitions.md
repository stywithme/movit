# 03 — تعريفات متغيرات وصف التمرين (RTx)

الإطار العام للوصف يتبع **FITT-VP**: Frequency, Intensity, Time, Type, Volume, Pattern, Progression.

الجدول التالي يعكس **Table 2** في البيان الأصلي (تعريفات تشغيلية مستخدمة في المراجعات المدخلة).

## Frequency

عدد الأيام في الأسبوع التي يُؤدى فيها تمرين المقاومة.

## Load (Intensity)

مقدار الوزن لكل تكرار؛ غالباً كنسبة من **1RM** أو كأقصى تكرارات ممكنة بتحميل معيّن (**RM**).

## Failure

إكمال التكرارات حتى **الفشل الإرادي** — عندما لا يعود بالإمكان إكمال الطور الانقباضي.

## Time under tension

الزمن الكلي لدعم الحمل؛ لكل تكرار = الزمن لإكمال تكرار واحد.

## Time of day

توقيت الجلسة (مثل صباحاً مقابل مساءً).

## Inter-set rest

الراحة بين المجموعات.

## Exercise order

ترتيب أنماط التدريب (مثل ترتيب تمارين مختلفة أو ترتيب المقاومة واللياقة الهوائية داخل الجلسة).

## Type — أمثلة

| النوع | التعريف المختصر |
|--------|------------------|
| Blood flow restriction (BFR) | تقليل تدفق الدم محلياً للعضلات العاملة |
| Free-weight RT | أثقال حرة تتحرك في الفضاء |
| Machine RT | أجهزة بمسار حركة ثابت نسبياً |
| Unstable RT | سطح غير ثابت (مثل كرة) |
| Variable load | تغيير الحمل أثناء التكرار |

## Contraction type

| المصطلح | المعنى |
|---------|--------|
| Eccentric overload | زيادة الحمل أو زمن التحت شد أثناء الطور اللاإرادي |
| Traditional | أداء متناسب للطورين الانقباضي واللاإرادي |

## Technique — أمثلة

| المصطلح | المعنى |
|---------|--------|
| Olympic-style weightlifting | حركات مثل الخطف والنتر |
| Partial ROM | مدى حركة غير كامل مقابل كامل للمفصل |
| Power RT | أداء الطور الانقباضي بأقصى سرعة إرادية ممكنة |

## Volume (sets)

عدد **المجموعات** لكل تمرين (مجموعة = تكرارات دون راحة بينها).

## Set structure — أمثلة

- **Drop sets:** إلى الفشل ثم تخفيض الحمل وراحة قصيرة بين المجموعات  
- **Cluster:** راحات مخططة داخل المجموعة  
- **Complex:** مجموعات ثقيلة ثم أخف  
- **Contrast:** تناوب ثقيل/خفيف بين المجموعات  
- **Rest redistribution:** راحات قصيرة بين كل تكرار  
- **Traditional:** مجموعة كاملة دون راحة داخلية حتى انتهاء التكرارات  

## Periodization

| النوع | الفكرة |
|--------|--------|
| Periodized | تغيير منظم لـ RTx عبر البرنامج |
| Block | كتل متعددة الأسابيع بأهداف مختلفة |
| Linear | زيادة الحمل وتخفيض الحجم تدريجياً |
| Nonperiodized | عدم تغيير منهجي لـ RTx |
| Undulating (nonlinear) | تغيير يومي أو أسبوعي لـ RTx |

---

**ملاحظة للمنتج:** واجهات ضبط «الجلسة» في التطبيق يمكن أن تعكس هذه المتغيرات كحقول اختيارية (load, sets, rest, ROM, intent/speed) مع توضيح أن التعريفات العلمية أعلاه هي مرجع ACSM 2026 وليست بالضرورة كل الحقول التي تستخدمها واجهتكم الحالية.
